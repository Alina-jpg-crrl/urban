HTML

  • 1.HTML (HyperText Markup Language) - это язык разметки, который используется для создания веб-страниц.
  • 2.Теги: HTML использует теги для определения структуры и содержания веб-страницы.
  • 3.Элементы: Теги образуют элементы, которые представляют собой блоки контента (например, абзац, заголовок, изображение)
  • 4.Атрибуты: Теги могут иметь атрибуты, которые предоставляют дополнительную информацию об элементе
  • 5.Структура: HTML-документ имеет четкую структуру

Дополнительные преимущества

  • 1:Простота: HTML относительно легко освоить.
  • 2:Широкая поддержка: HTML поддерживается всеми популярными браузерами.
  • 3:Открытый стандарт: HTML является открытым стандартом, доступным для всех.

CSS

  • 1:CSS (Cascading Style Sheets) — это язык стилей, который используется для оформления веб-страниц.
  • 2:Селекторы: CSS использует селекторы для определения, к каким элементам HTML должны применяться стили.
  • 3:Свойства: Свойства определяют различные аспекты стиля
  • 4:Значения: Свойствам присваиваются значения, которые определяют стиль элемента
  • 5:Каскадность: CSS имеет систему каскадности, которая определяет приоритетность стилей в случае конфликтов.
  • 6:Встроенные стили: Стиль можно задать непосредственно внутри элемента HTML
  • 7:Внутренние стили: Стиль можно определить внутри HTML-документа.
  • 8:Внешние стили: Стиль можно определить в отдельном файле CSS, который подключается к HTML-документу
  • 9:Медиа-запросы: CSS позволяет создавать стили, которые применяются только при определенных условиях, например, при определенном размере экрана или ориентации.
  • 10:Препроцессоры CSS: Препроцессоры CSS, такие как Sass и Less, добавляют дополнительные возможности, такие как переменные, функции и вложенность, которые упрощают создание и поддержку кода CSS.

Дополнительные преимущества

  • 1:Улучшает внешний вид: CSS позволяет создавать привлекательные и современные веб-страницы.
  • 2:Повышает доступность: CSS помогает сделать веб-страницы доступными для пользователей с ограниченными возможностями.
  • 3:Сокращает время разработки: CSS позволяет повторно использовать стили, что сокращает время разработки.
  • 4:Улучшает производительность: CSS может быть оптимизирован для повышения производительности веб-страниц

JavaScript

  • 1:JavaScript (JS) — это язык программирования, который используется для добавления интерактивности и динамики на веб-страницы.
  • 2:Клиентская сторона: JavaScript, как правило, выполняется в браузере пользователя на клиентской стороне, делая веб-страницы интерактивными.
  • 3:Синтаксис: JS имеет синтаксис, основанный на C-подобных языках, с использованием фигурных скобок, точек с запятой и ключевых слов.
  • 4:Переменные: JavaScript использует переменные для хранения данных.
  • 5:Типы данных: JS поддерживает различные типы данных
  • 6:Операторы: JS использует операторы для выполнения различных операций
  • 7:Условные операторы: if, else, else if позволяют выполнять различные блоки кода в зависимости от определенных условий.
  • 8:Циклы: for, while, do...while позволяют выполнять блоки кода несколько раз
  • 9:Функции: JS использует функции для группирования кода, что делает его более организованным и повторно используемым.
  • 10:DOM (Document Object Model): JS использует DOM для доступа и изменения элементов HTML-документа.

Дополнительные преимущества

  • 1:Интерактивность: JS позволяет сделать веб-страницы более интерактивными и привлекательными для пользователей.
  • 2:Динамичность: JS позволяет изменять содержимое и стиль веб-страниц динамически.
  • 3:Всесторонность: JS используется не только для веб-разработки, но и для разработки мобильных приложений, игр, серверных приложений и т.д.
